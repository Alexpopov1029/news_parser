# Парсер новостей

Простой Python-скрипт, который собирает заголовки новостей с сайта Hacker News
и сохраняет их в CSV-файл.

## Использование
1. Установите библиотеки:
   ```bash
   pip install -r requirements.txt
   ```

2. Запустите скрипт:
   ```bash
   python main.py
   ```

3. Результат будет сохранён в `news.csv`

## Технологии
- Python
- requests
- BeautifulSoup
